# Flask App 

Create  HTTP Server. 


## GET /products 
    get a list of products


## POST /products 
    Create the Product 

## PATCH /products/:id 
    update the product with the given payload 

## DELETE /products/:id 
    delete product with that id. 

## Login 
    Send Email and Password , if correct you will get a token. 

### NOTE: Send token in all requests.
